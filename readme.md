### Particle of Standard Asset, for Visual Effect Graph.

Unity 2018.X以前で利用可能だった、Standard Assetに含まれるParticle System(Shuriken)のサンプルを  
VFX Graphで同じ動きとなるように作成しました。

一部製作中ですが、習作などでお使いください。
