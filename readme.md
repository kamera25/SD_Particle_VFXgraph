## Particles of Standard Asset were remade using Visual Effect Graph.

Unity 2018.X以前で利用可能だった、[Standard Asset]()に含まれるParticle System(Shuriken)のサンプルを  
VFX Graphで同じ動きとなるように作成しました。

### 含まれているパーティクル - Available particles

 * Afterbunner
 * DustStorm
 * ExplosionMobile
 * FireComplex
 * Fireworks
 * Flare
 * HouseMobile
 * Smoke
 * Steam
 * TyreBurnoutSmoke
 * WildFire
